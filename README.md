# Gwangjin-Gu
광진구 빅데이터 공모전
![image](https://github.com/GaGyeong-Kim/Gwangjin-Gu/assets/108977032/92b11148-1b18-4275-9d13-10f294f69612)


### 분석자료
- 광진구 주정차단속현황
- 광진구 공영 주차장
- 광진구 인구수
- 동별 공영주차장, 인구, 자동차

### 전처리
- 정규표현식을 활용한 단속동 컬럼 전처리
- 단속일시 컬럼 형식 변경(datetime)
- 동명 범주화(빈도수 계산 용이하게 하기 위함)
- 행정기관 그룹화
- 남/여 인구수 전처리
